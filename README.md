# VLMs Are Blind — Research Notes

Research workspace exploring the limitations of Vision Language Models (VLMs) in visual perception tasks, inspired by ["Vision Language Models Are Blind" (Rahmanzadehgervi et al., 2024)](https://arxiv.org/abs/2407.06581).

## What This Is

A collection of notes, paper summaries, and analysis investigating how and why state-of-the-art VLMs fail at basic visual tasks that humans find trivial — such as counting intersecting lines, identifying overlapping shapes, or reading text in images.

## Structure

```
papers/          # Source PDFs
notes/           # Summaries, analysis, brainstorming
specs/           # Research plans and investigation outlines
src/             # Any supporting scripts (visualization, analysis)
```

## Key Questions

- What specific visual tasks do VLMs consistently fail at?
- Are these failures fundamental architectural limitations or training data gaps?
- How do different VLM architectures compare on these tasks?
- What approaches might address these blind spots?

## Core Paper

**Vision Language Models Are Blind**
