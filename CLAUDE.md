# VLMs Are Blind — Research Project

## Purpose
This is a research workspace for investigating the limitations of Vision Language Models (VLMs), anchored by the paper "Vision Language Models Are Blind" (arXiv:2407.06581). The focus is on literature review, brainstorming, and analysis — not primarily a software project.

## Workflow
- **Web search & paper analysis** are the primary activities
- **Summaries and notes** go in `notes/`
- **Specs/plans** go in `specs/` per global conventions
- **Papers** (PDFs) go in `papers/`
- **Code** (if any) goes in `src/` — likely small scripts for visualization or data analysis

## Conventions
- When summarizing papers, include: citation, key claims, methodology, results, and relevance to the "VLMs are blind" thesis
- Use Markdown for all notes and summaries
- Keep a running bibliography in `notes/bibliography.md`

## Git
- This repo does not require a remote origin — it's a local research workspace
- Still use git locally for history tracking
